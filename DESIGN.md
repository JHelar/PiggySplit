### High level user flow

Following is a high level overview of the user flow

1. User sign in with email code token
2. New users enters credentials to identify the user when within a group
3. User creates group
4. User invites other members via shared deep link to group
5. Members enters expenses
6. When members has entered expenses, all manually flag themselves as ready
7. When all in group is ready, split expenses evenly throughout the group, setup optimal payments so that members has to pay as few members as possible
8. When all has payed, mark group as resolved
